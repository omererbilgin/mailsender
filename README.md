# Mail Sender

A simple program that sends a mail to all the e-mail adresses from a list by Omer Erbilgin

# Installation

Tested on Ubuntu 20.04

Requirements:
            gcc
            python3

$ git clone https://github.com/omererbilgin/mailsender
$ cd mailsender
$ gcc listset.c 
$ ./a.out
